# PWA Icons Placeholder

Иконки для PWA (icon-192.png и icon-512.png) нужно добавить позже.

Для первого деплоя это не критично - приложение будет работать, но без иконок в манифесте.

## Как создать иконки:

1. Создать логотип Go2Asia в размерах 192x192 и 512x512
2. Сохранить как PNG с прозрачным фоном
3. Разместить в `apps/go2asia-pwa-shell/public/`:
   - `icon-192.png`
   - `icon-512.png`
