# ADR 0005: Cloudflare как глобальный edge-слой для Go2Asia

## Статус
Принято

## Дата решения
2025-12-10

## Контекст
Пользователи Go2Asia находятся в:
- России,
- Вьетнаме,
- Таиланде,
- на Фукуоке,
- путешествуют между регионами.

Нужна высокая доступность и защита.

## Решение
Cloudflare становится **единой точкой входа**:
- DNS,
- CDN,
- Edge-cache,
- WAF,
- Rate Limiting,
- API Gateway (в будущем),
- Zero Trust для внутренних сервисов.

## Последствия
- улучшение доступности,
- защита от блокировок,
- снижение нагрузки на backend,
- ускорение загрузки статики.

## Альтернативы
AWS CloudFront → хуже в РФ и ЮВА.  
Netlify CDN → не покрывает нужные зоны.
