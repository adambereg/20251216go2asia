# ADR 0008: Двухконтурная токеномика Go2Asia (off-chain + on-chain TON)

## Статус
Принято

## Дата решения
2025-12-10

## Контекст
Go2Asia использует:
- Points — soft currency,
- G2A Token — on-chain currency,
- NFT — бейджи, достижения, коллекции.

Нужно избежать прямой работы backend с приватными ключами и снизить юридические риски.

## Решение
Вводится двухконтурная модель:

1. **Off-chain Token Service**  
   — хранит балансы  
   — начисляет токены  
   — ведёт историю операций  
   — проводит бизнес-логику  

2. **Blockchain Gateway / Wallet Service**  
   — обрабатывает связь с TON  
   — вывод токенов  
   — депозиты  
   — mint NFT  
   — работает через master-wallet  
   — никогда не передаёт приватные ключи наружу  

## Последствия
- безопасность,
- auditability,
- возможность ввести DAO,
- масштабируемость.

## Альтернативы
Хранить все токены on-chain → дорого и медленно.
