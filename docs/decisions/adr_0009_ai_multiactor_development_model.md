# ADR 0009: Мультиагентная модель разработки Go2Asia (AI Actors)

## Статус
Принято

## Дата решения
2025-12-10

## Контекст
Go2Asia создаётся в среде, где:
- значительная часть кода генерируется AI,
- используются роли агентов (Оркестратор, Аналитик, Архитектор, Планировщик, Dev, QA, Security, Tech Writer),
- важна согласованность решений.

## Решение
Принять модель **AI Multi-Actor Development**, где:

- каждый агент имеет свой системный промт,
- решения принимаются через Orchestrator,
- архитектура фиксируется в ADR и docs/,
- Cursor работает с "капсулами" — папками контекста.

## Последствия
- согласованное развитие,
- отсутствие противоречий между агентами,
- автоматизация больших объёмов разработки,
- воспроизводимость.

## Альтернативы
Единый AI без распределения ролей → хаос и непредсказуемые изменения.
