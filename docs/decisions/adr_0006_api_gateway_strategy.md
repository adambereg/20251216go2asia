# ADR 0006: Стратегия API Gateway

## Статус
Принято

## Дата решения
2025-12-10

## Контекст
Многие микросервисы должны быть доступны через единый слой:
- Auth
- User
- Guru Nearby
- Atlas, Pulse, Blog
- Points, NFT, Blockchain Gateway
- Notification
- Rielt, RF Partners

Нельзя отдавать прямые публичные ссылки на микросервисы.

## Решение
API Gateway на Cloudflare Workers.

Функции:
- маршрутизация,
- авторизация,
- rate limiting,
- кэширование,
- проверка токенов,
- защита.

## Последствия
Гибкость маршрутов и единая интеграция фронтенда.

## Альтернативы
Kong / Traefik → слишком тяжёлые для edge-first архитектуры.
