# ADR 0002: Выбор Drizzle ORM как основного инструмента работы с БД

## Статус
Принято

## Дата решения
2025-12-10

## Контекст
Go2Asia использует PostgreSQL как основную БД.  
Требования:
- типобезопасность,
- миграции в репозитории,
- высокая производительность,
- интеграция с Edge/Serverless окружениями (Cloudflare Workers, Netlify Functions),
- предсказуемый SQL.

Были рассмотрены Prisma, TypeORM, Kysely.

## Решение
Выбрать **Drizzle ORM** в качестве основной ORM.

## Причины
- минималистичный и быстрый SQL-first подход,
- лучший DX для Typescript,
- поддержка edge-runtime,
- миграции без overhead,
- нет скрытого поведения (как в Prisma),
- идеально подходит для микросервисов.

## Последствия
Платформа получает надёжный и лёгкий слой работы с БД.  
Миграции становятся предсказуемыми и легко управляемыми CI/CD.

## Альтернативы
- Prisma → слишком тяжёлый слой, плохо подходит для edge.
