# Layouts: Module Layouts — шаблоны страниц для модулей Go2Asia

Каждый модуль должен использовать один из стандартных layouts.

---

# 1. Основные layouts

## 1.1. List Layout (Atlas, Pulse, Blog, RF)
Содержит:
- hero,
- поисковая строка,
- фильтры (mobile: accordion),
- карточки в сетке или списке.

## 1.2. Detail Layout (Article, Place, Event, Listing)
Содержит:
- hero,
- блок мета-данных,
- основной контент,
- карточки похожего контента.

## 1.3. Search First Layout (Guru, Rielt)
Содержит:
- большая строка поиска,
- фильтры,
- список или сетка результатов.

---

# 2. Адаптивность

Mobile:
- одна колонка,
- filters → accordion,
- CTA-кнопки 100% ширины.

Desktop:
- две или три колонки,
- filters слева,
- контент + правый сайдбар (по необходимости).

---

# 3. Ограничения для AI-разработчиков

- Нельзя создавать новый layout без согласования.
- Все модули должны наследовать один из существующих.
- Новые компоненты допускаются только на базе существующих токенов.
