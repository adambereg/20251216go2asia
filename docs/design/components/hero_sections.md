# Components: Hero Sections — hero-блоки страниц

Hero — крупный верхний блок страницы, задающий атмосферу модуля.

---

# 1. Типы hero-блоков

## 1.1. Simple Hero (Atlas, Blog)
- Заголовок (H1),
- Короткое описание,
- Фоновый цвет или лёгкая текстура,
- Без изображения или с маленьким иллюстративным элементом.

## 1.2. Image Hero (Pulse, Atlas Places)
- Большое изображение (16:9 или 21:9),
- Градиентная подложка для текста,
- Элементы:
  - название,
  - локация,
  - категория.

## 1.3. Functional Hero (Rielt, RF)
- Заголовок,
- Поисковая строка,
- Фильтры,
- CTA-кнопка.

---

# 2. Правила изображений

- изображение всегда имеет круглые скругления,
- обязательная обводка (`1px solid var(--color-neutral-300)`),
- изображение не должно быть слишком тёмным или контрастным.

---

# 3. Адаптивность

Mobile:
- текст центрирован,
- высота 200–240px.

Desktop:
- текст слева,
- высота 320–480px.
