# Components: Cards — карточки контента Go2Asia

Карточки — основа визуального отображения данных во всех модулях.

---

# 1. Общие принципы

- карточки имеют рамку + скругление,
- изображение всегда с border-radius,
- тень слабая или отсутствует,
- пропорция фото зависит от типа карточки,
- весь кликабельный блок — карточка целиком.

---

# 2. Типы карточек

## 2.1. Standard Card (Atlas, RF, Guru)
- Изображение: 16:9,
- Заголовок: H3,
- Подзаголовок: категория, страна/город.

## 2.2. Event Card (Pulse)
- Изображение: 4:3,
- Элементы:
  - дата,
  - место,
  - статус (free/paid).

## 2.3. Article Card (Blog)
- Изображение: 16:9,
- Элементы:
  - название,
  - категория,
  - дата публикации.

## 2.4. Listing Card (Rielt)
- Изображение: 1:1,
- Элементы:
  - цена,
  - локация,
  - параметры (кв.м, комнаты),
  - бейджи (новое, супер-цена).

---

# 3. Hover/Active поведение

Desktop:
- небольшое поднятие (`transform: translateY(-2px)`),
- тень увеличивается.

Mobile:
- нет hover,
- увеличенная зона клика.

---

# 4. Обязательные поля

AI-агенты **не должны создавать карточки без картинки, заголовка и категории**.
