# Components: Forms — формы ввода Go2Asia

Формы должны быть простыми, минималистичными и удобными для мобильного ввода.

---

# 1. Поля ввода

## 1.1. Text Input
- рамка: `1px solid var(--color-neutral-300)`
- внутренняя тень запрещена
- радиус: `var(--radius-md)`
- иконки слева/справа по необходимости

## 1.2. Select
- стрелка справа,
- одинаковая высота с Text Input,
- список — нативный или кастомный.

## 1.3. Date/Time Picker
- единый стиль на всех модулях.

---

# 2. Валидация

Ошибки:
- красная подсветка рамки,
- текст ошибки под полем,
- иконка предупреждения.

Успех:
- зелёная рамка (не обязательна).

---

# 3. Кнопки отправки

- Primary button — зелёная,
- 100% ширины на мобильном.

---

# 4. Согласованность

Cursor MUST:
- использовать токены,
- не придумывать собственных стилей,
- не добавлять нестандартные элементы.
