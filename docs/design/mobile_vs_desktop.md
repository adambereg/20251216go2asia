# Mobile vs Desktop — адаптивная стратегия Go2Asia

Основной принцип Go2Asia: **Mobile-first**, оптимизация под Desktop — вторым этапом.

---

# 1. Breakpoints

```
--breakpoint-sm: 480px;
--breakpoint-md: 768px;
--breakpoint-lg: 1024px;
--breakpoint-xl: 1440px;
```

---

# 2. Отличия мобильной версии

## Mobile:
- нижняя навигация (Bottom Bar),
- свернутые фильтры (accordion),
- карточки в 1 колонку,
- упрощённые hero-блоки,
- модальное поведение поиска.

## Desktop:
- боковая навигация (Sidebar),
- фильтры слева,
- карточки в 2–4 колонках,
- расширенные hero-блоки,
- поиск в header.

---

# 3. Принцип читабельности

Мобильная версия использует:

- крупные кнопки,
- большие отступы,
- упрощённую типографику.

Desktop расширяет интерфейс, сохраняя структуру.

---

# 4. Модульная согласованность

Каждый модуль должен:

- использовать одинаковые брейкпоинты,
- соблюдать единые layout-паттерны,
- не вводить уникальные адаптивные правила без согласования.
