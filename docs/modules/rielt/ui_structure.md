# Rielt.Market Asia — UI Structure  
(основано на реальных UI-наработках Cursor + дизайн-системе Go2Asia)

---

# 1. Общие принципы дизайна модуля

- Цветовая тема: **зелёный брендовый градиент Rielt.Market** (как на скрине).  
- Компоненты наследуют общие паттерны Go2Asia:
  - ModuleHero (как в Atlas/Pulse/Blog),
  - мягкие карточки (soft-card),
  - чёткая иерархия текстов,
  - компактные чипы-фильтры,
  - большие галереи изображений,
  - двухколоночные layout для детальных страниц.
- Главный UX-фокус:
  - быстрый поиск жилья,
  - доступные фильтры,
  - удобная карточка объекта,
  - мгновенный CTA (“Отправить запрос”).

---

# 2. Главная страница каталога — `/rielt`

## 2.1. Hero-блок (ModuleHero)

**Структура:**
- Заголовок: `Rielt.Market`
- Подзаголовок: «Поиск краткосрочной и долгосрочной аренды жилья в Юго-Восточной Азии»
- Фоновый зелёный градиент (100% ширины)
- Иконка модуля (как на скрине)

**Эффект:**
- Функция навигационного якоря: пользователь сразу понимает, где находится.

---

# 3. Поисковый блок (SearchPanel)

Располагается сразу под Hero.

## 3.1. Режимы аренды (switch)
- **Краткосрочно**
- **Долгосрочно**
(раздельные пресеты фильтров + разные формы расчёта стоимости)

## 3.2. Поля поиска:
- Поле ввода: «Куда?»  
  - autocomplete по городам, районам (интеграция Atlas)
- Дата заезда / дата выезда (для краткосрочной)
- Кол-во гостей  
- Кнопка CTA: **Найти жильё**

## 3.3. Дополнительные фильтры (чипы):
- Только RF  
- Проверено PRO  
- Сегодня заселение  
- Мгновенное бронирование  
- Цена  
- Удобства (в будущем)

Эти чипы работают как фильтры 1-го уровня.

---

# 4. Блок “Популярно сейчас”

Горизонтальный список карточек **CityCard**:

Каждая карточка содержит:
- фоновое изображение города,
- название,
- количество актуальных объявлений,
- чип страны.

Клик перенаправляет на страницу города: `/rielt/city/{slug}`.

---

# 5. Подборки редакции (“Editorial picks”)

Каждый раздел включает:
- заголовок (“Для семей”, “Для цифровых кочевников”, “Долгосрочная аренда”),
- подзаголовок-пояснение,
- 1–4 карточки недвижимости.

Компонент карточки — **PropertyFeaturedCard**.

Структура карточки:
- Обложка (16:9, rounded)
- Название: «Вилла у пляжа с бассейном»
- Локация (город, страна)
- Рейтинг (звезды + кол-во отзывов)
- Иконки гостевых параметров (гости, спальни, кровати, ванные)
- Цена: `$1200`
- Примечание (например: «Месяц» или «Ночь»)

# 6. Страница списка объявлений города / результата поиска

## 6.1. Заголовок
- Название города или “Результаты поиска”
- Количество найденных объектов

## 6.2. Фильтры второго уровня
- Цена (ползунок / диапазон)
- Тип жилья: Квартира / Дом / Комната / Вилла / Co-living
- Удобства: WiFi, кухня, бассейн, стиралка, кондиционер, парковка
- Сложность (для PRO-аналитики)
- RF / PRO / Verified
- Сортировка:
  - по популярности  
  - по цене  
  - по новизне  

## 6.3. Варианты отображения списка:
- **Сетка** (карточки)
- **Список**
- **Карта + Список (будущее)**

## 6.4. Карточка объекта — PropertyCard

Элементы карточки:
- Галерея (до 3 фото)
- Бейджи: Новое / PRO / RF / Verified
- Название
- Локация (Город, Район)
- Параметры:
  - гости  
  - спальни  
  - кровати  
  - ванные  
  - площадь  
- Цена (за ночь / месяц, в зависимости от режима)
- CTA: сохранить (иконка сердечка)

# 7. Страница объекта — `/rielt/property/{id}`

Макет: **Hero-галерея сверху → два столбца**

## 7.1. Hero Image Gallery
- Большое фото сверху (16:9), свайп/стрелки
- Мини-галерея 3 превью снизу
- Бейдж “Новое” или RF

## 7.2. Левая колонка — Информация о жилье

### (A) Название + Локация
- Заголовок: «Просторные апартаменты с видом на город»
- Город, район, страна  
- Рейтинг: 4.8 (37 отзывов)

### (B) Характеристики
Сетка из иконок:
- Гости
- Спальни
- Кровати
- Ванные
- Площадь

### (C) Стоимость
- `$1200 / месяц`
- Депозит
- Минимальный срок (30 ночей)

### (D) Описание
Полный текст с форматированием:
- особенности жилья,
- преимущества,
- окружение,
- инфраструктура.

### (E) Удобства
Грид 2×N:
- WiFi  
- Кондиционер  
- Бассейн  
- Кухня  
- TV  
- Парковка  
- Прачечная  
и т.д.

### (F) Расположение
Интерактивная карта (интеграция с Guru):
- маркер объекта,
- окружение (кафе, магазины, пляжи, станции).

---

# 7.3. Правая колонка — CTA блок

## (A) Цена и основные условия
- `$1200 / месяц`
- Дополнительные условия (check-in, deposit)

## (B) CTA кнопки:
- **Отправить запрос** (главная кнопка)

# 11. Адаптивность

## Мобильная версия:
- Сетка карточек → 1 в ряд,
- Поисковый блок сворачивается в accordion,
- CTA кнопки фиксируются снизу,
- Галерея превращается в swipe-слайдер,
- Левая/правая колонка → вертикальный стек.

- 3D-туры (при наличии)
- Встроенный чат арендатор–арендодатель
- AI-оценка правдивости объявления
