# Rielt.Market Asia — Обзор модуля

## 1. Назначение модуля
Rielt.Market Asia — это универсальная площадка по недвижимости ЮВА, ориентированная преимущественно на русскоязычных экспатов и путешественников. Модуль позволяет:
- искать долгосрочную аренду,
- покупать и продавать недвижимость,
- размещать объявления арендодателям и агентам,
- обеспечивать верификацию объектов и пользователей,
- использовать AI для подбора вариантов, анализа объявлений и персональных рекомендаций,
- интегрироваться с Connect Asia (реферальная сеть), Guru Asia (места рядом), Atlas Asia (районы и города).

## 2. Основные пользовательские сценарии
### Путешественник / Экспат
- Ищет квартиру или дом на долгий срок.
- Сравнивает цены, районы, удобства.
- Получает AI-рекомендации.
- Сохраняет избранное.
- Связывается с арендодателем / агентом.

### Арендодатель / Агент
- Размещает объявления.
- Управляет объектами, доступностью, ценами.
- Получает лиды и аналитику.
- Проходит верификацию.

### PRO / Партнёр
- Может получать награды за приведённых арендаторов.
- Может продвигать объявления (Boost).

## 3. Главная ценность модуля
- Прозрачность рынка недвижимости ЮВА.
- Единая точка входа для арендаторов и инвесторов.
- Надёжность через верификацию.
- Умный поиск и AI-анализ.

## 4. Связи с другими модулями
- **Atlas Asia** — районы, общая инфраструктура, география.
- **Space Asia** — публикации о жилье, отзывы.
- **Pulse Asia** — события возле объекта.
- **Guru Asia** — места рядом.
- **Connect Asia** — реферальные бонусы.
- **Tokenomics** — Points и NFT для активности.
