# Atlas Asia — UI Structure  

Atlas Asia — это живой вики-справочник стран ЮВА, объединяющий структурированный контент, интерфейсы фильтрации и глубокую навигацию.  
UI модуля полностью основан на реальных макетах, созданных Cursor, и оптимизирован для PWA.

---

Atlas Asia живёт внутри общего PWA-shell Go2Asia.

# 1. Главная страница Atlas Asia  
**URL:** `/atlas`

## 1.1. Hero-секция
- большой горизонтальный баннер;
- слева — иконка «глобус» + заголовок **Atlas Asia**;
- подзаголовок: «Живой вики-справочник по странам Юго-Восточной Азии с UGC и редакционной поддержкой».

## 1.2. Верхние табы (основная навигация)
- **Страны**
- **Города**
- **Места**
- **Гайды**
- **Темы**

Табы всегда фиксированы на всех страницах модуля.

## 1.3. Поисковая строка
Универсальный поиск: страна, город, место, гайд.

## 1.4. Сетка стран
Карточка страны отображает:
- флаг (или код страны в углу карточки);
- изображение;
- название страны;
- описание (1 предложение);
- метрики: количество городов / количество мест.

Количество карточек на строку: **3 (desktop)** / **2 (tablet)** / **1 (mobile)**.

# 11. Общие UI-принципы Atlas Asia

## 11.1. Шаблон структуры страницы
Каждая страница следует паттерну:

**Hero → Sidebar → Content**

## 11.2. Компоненты дизайна
- карточки мест / городов / стран;
- теги / бейджи;
- фиксированный sidebar;
- крупные фотографии;
- метрики просмотров / обновлений;
- блоки «Проверено редакцией».

## 11.3. Взаимосвязь модулей
Atlas взаимодействует с:
- Pulse (события рядом);  
- Russian Friendly (партнёрские предложения);  
- Quest (квесты, связанные с местами);  
- Space (UGC-публикации с привязкой к локациям).

---

# 12. Адаптивность
Desktop:
- sidebar слева, фиксирован  
Mobile:
- sidebar превращается в выпадающее меню  
Tablet:
- две карточки в ряд
