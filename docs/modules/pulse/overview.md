Pulse Asia — Overview
Статус: Core Module
Назначение: единый календарь событий и мероприятий Юго-Восточной Азии, объединяющий UGC, партнёрские мероприятия и редакционные подборки.
Pulse Asia — это «пульс жизни региона»: концерты, фестивали, встречи экспатов, бизнес-мероприятия, экскурсии, спортивные события, локальные активности, мастер-классы.
Модуль предоставляет многоуровневый интерфейс просмотра событий (месяц → неделя → день → список) и глубокую карточку мероприятия.
________________________________________
Цели модуля
1. Дать пользователю удобный календарь жизни региона
•	Быстрый осмотр: что происходит в этом месяце.
•	Детальная навигация по неделям / дням.
•	Фильтрация по странам → городам → категориям.
2. Повысить вовлеченность через социальную механику
•	RSVP (интересуюсь / пойду).
•	Сохранения (bookmark).
•	Короткие обзоры и отчёты от сообщества (через Reactions).
•	Связка с Space Asia: отчёты и публикации участников.
3. Поддержать партнёрскую модель Russian Friendly
•	Мероприятия бизнес-партнёров помечаются ярлыком «Проверено».
•	Возможность интеграции ваучеров и премиум-ваучеров.
4. Стать входной точкой для Guru Asia
•	Из Pulse → Guru: «Похожие события рядом с вами».
•	Уведомления о мероприятиях поблизости.

Основная концепция UI Pulse Asia
Pulse Asia построен вокруг четырёх режимов просмотра:
Режим	Назначение
Месяц	Обзор месяца, распределение событий, цветные точки-категории
Неделя	Планирование на неделю, подробный список событий
День	Полная афиша конкретного дня
Список	Лента мероприятий с фильтрами и поиском
Все режимы используют один набор фильтров:
•	страна → город → место
•	категория
•	типы (онлайн / оффлайн)
•	UGC / редакция / партнёр
Поиск работает как полнотекстовый по: название, организатор, тег, локация.

Основные страницы Pulse Asia
1. Стартовая страница — Календарь месяца
URL: /pulse
Элементы:
•	Хиро-блок с названием модуля и подзаголовком
•	Переключатели: Месяц / Неделя / День / Список
•	Навигация по месяцам
•	Универсальная строка поиска
•	Фильтры стран, городов, мест
•	Фильтры категорий (еда, культура, музыка, спорт, IT, сообщество, детское, ночная жизнь)
•	Календарная сетка:
o	каждая дата содержит цветные точки — по количеству событий категории
o	клик открывает дневной список
Снизу — секция «События этого дня».
________________________________________
2. Страница недели
URL: /pulse/week/2025-11-02
Элементы:
•	Полоска дней недели
•	Для каждого дня: количество событий + превью точек категорий
•	Лента событий на всю неделю
•	Событие отображается карточкой (партнёр / проверено / UGC / рейтинг / цена / время / место)
________________________________________
3. Страница дня
URL: /pulse/day/2025-11-09
Элементы:
•	Большой блок с датой
•	Переключатели предыдущий / следующий день
•	Полоска мини-календаря на 7 дней
•	Лента всех событий дня
•	Возможность быстро переключаться по дням
________________________________________
4. Страница списка (лента событий)
URL: /pulse/list
Элементы:
•	Полный набор фильтров
•	Сортировки: ближайшие / популярные / бесплатные
•	Стандартные карточки мероприятий
Используется как основной инструмент поиска.
________________________________________
Карточка события
URL: /pulse/event/{id}
Карточка — одна из самых детальных в экосистеме.
Состав блоков:
1. Хиро-блок
•	Большая фотография
•	Категории (еда, культура, спорт…)
•	Страна / город / место
•	Метаданные: проверено / редакция / UGC
•	Кнопки: RSVP, Сохранить, Репост (через Reactions)
2. О событии
•	Описание
•	Расписание и дата
•	Время начала и окончания
•	Цена билета / бесплатное
•	Тэги
3. Локация
•	Город
•	Место
•	Организатор и его рейтинг
•	Карта (будет добавлена в следующей фазе)
4. Блок "Отчёты сообщества"
Интеграция через Reactions Service:
•	short_review
•	рейтинг
•	репосты-отчёты
•	треды вопросов организатору
5. Похожие события
•	алгоритм через Content Recommendations / Guru Asia

Интеграции Pulse Asia с другими модулями
✔ Reactions Service
Pulse отправляет события:
•	event.rsvp
•	event.saved
•	event.review.created
•	event.thread.reply
✔ Connect Service
События Pulse могут давать награды:
•	посещение партнёрского мероприятия → Points
•	создание отчёта → Points
•	UGC событие, опубликованное PRO → NFT бейджи
✔ RF / Voucher Service
•	мероприятия бизнес-партнёров могут включать ваучеры
•	PRO за продвижение может получать Points
✔ Atlas Asia
•	на странице города отображаются ближайшие события
•	в карточке события отображается информация о локации
✔ Guru Asia
•	гео-уведомления: «рядом с вами сегодня 3 события»
•	персональные рекомендации по интересам
