# Russian Friendly Asia — Overview

Russian Friendly Asia — модуль экосистемы Go2Asia, объединяющий дружественные туристические сервисы, кафе, рестораны, магазины, коворкинги, сервисы услуг и локальные компании, ориентированные на русскоязычных туристов и экспатов.

Модуль состоит из трёх уровней:

1. **Публичный каталог**  
   • Каталог партнёров RF  
   • Каталог ваучеров  
   • Страница заведения  
   • Страница ваучера  
   • Подборки PRO (как в Rielt.Market)

2. **Кабинет Партнёра (Business Dashboard)**  
   • Профиль заведения  
   • Управление ваучерами  
   • Управление отзывами  
   • Статистика  
   • Статус проверки PRO

3. **Кабинет PRO-куратора (PRO Dashboard)**  
   • Мои партнёры  
   • Онбординг бизнеса  
   • Проверка бизнеса  
   • Вознаграждения Points/G2A  
   • Дашборд эффективности PRO

---

## Цели модуля

- Создать единый каталог проверенных мест, дружественных русскоязычным путешественникам.
- Дать бизнесам инструмент продвижения: ваучеры, акции, рейтинги, PRO-поддержку.
- Сформировать рынок PRO-кураторов — локальных представителей Go2Asia.
- Интегрировать модуль с токеномикой (Points, G2A), рейтингами и Space Asia.

---

## Основные функции

### Для пользователей
- Поиск мест по категориям, городам, фильтрам (RF-статус, PRO-проверено).
- Просмотр подробных карточек заведений с фото, меню/услугами, графиком работы.
- Покупка или получение ваучеров.
- Оставление отзывов.
- Подборки редакции / подборки PRO.

### Для партнёров (заведения)
- Управление профилем.
- Создание акций и ваучеров.
- Ответы на отзывы.
- Ведение статистики.
- Получение статуса PRO-проверено.

### Для PRO-кураторов
- Добавление новых бизнесов.
- Онбординг и проверка.
- Отчёты о проверках.
- Получение вознаграждений.
- Дашборд эффективности.

---

## Связи с другими модулями

- **Atlas Asia** — локации заведений (привязка к городам/районам).  
- **Pulse Asia** — события, проводимые партнёрами.  
- **Space Asia** — отзывы, публикации, соц-профили.  
- **Token Service** — Points/G2A за активности.  
- **PRO Service** — уровни PRO, управление кураторами.
