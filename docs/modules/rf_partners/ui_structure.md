# Russian Friendly Asia — UI Structure

---

# 1. Публичные страницы

## 1.1. Главная RF Catalog
Как на скриншоте:
- Hero блок (синий)
- Верхнее меню: Каталог партнёров | Каталог ваучеров | Кабинет партнёра | PRO Dashboard
- Фильтры категорий (кафе, рестораны, магазины...)
- Фильтры статусов: RF, PRO, скидки, подарки
- Сетка карточек заведений
- Подборки PRO (горизонтальные секции)
- Пагинация

### Карточка партнёра:
- Фото + галерея
- Название
- Категория и город
- Значки RF и PRO
- Рейтинг и количество отзывов
- Основные теги
- Кнопка «Подробнее»

---

## 1.2. Страница заведения
(точно как на скриншоте)

Секции:
1. Hero галерея
2. Название + статус (RF / PRO)
3. Основная информация
4. Контакты
5. Адрес + карта
6. Активные ваучеры
7. RF стандарт (чеклист)
8. Социальные сигналы и UGC

Примечание: RF не реализует inline-комментарии “под карточкой”. Раздел “Отзывы/UGC” отображает:
- агрегаты (рейтинг/счётчики),
- ссылки на UGC‑посты/репосты в Space, связанные с партнёром,
- CTA “Поделиться” → репост партнёра в Space (обсуждение в контексте репоста).

Правый сайдбар:
- Получить ваучер
- Поделиться
- Статистика (для владельца)

---

## 1.3. Каталог ваучеров
Секция:
- Фильтры: скидка / бонус / подарок / категории
- Карточка ваучера (фото, % скидки, условия, кнопка «Получить»)

# 2. Кабинет Партнёра

## 2.1 Dashboard
Отображает:
- Просмотры за месяц
- Получено ваучеров
- Активно ваучеров
- Рейтинг

Быстрые действия:
- Создать ваучер
- Управление ваучерами
- Ответы на отзывы
- Статистика

---

## 2.2 Профиль заведения
Разделы:
- Основная информация
- Категория
- Описание
- Контакты
- Адрес
- График работы

---

## 2.3 Ваучеры партнёра
- Список ваучеров
- Статусы: активен / архив
- Кнопки: редактировать / архивировать
- Блок погашения ваучера (ввод кода)

---

## 2.4 Отзывы
- Список отзывов
- Ответы партнёра (через social-контур Space / треды общения; без inline-комментариев “под карточкой”)

Дополнение:
- В “Отзывы” попадают UGC‑сигналы из Space/Reactions (короткие отзывы/рейтинги/репосты).

---

## 2.5 Статистика
- Графики ваучеров
- Статистика посещений
- История действий

# 4. VIP (Consumer) Surface

RF — multi-sided модуль. Помимо публичного каталога, VIP‑пользователь имеет “операционные” сценарии:

## 4.1 Мои ваучеры
- Список полученных/купленных ваучеров
- Статусы: активен / использован / истёк
- QR/код для предъявления партнёру

## 4.2 Квесты с RF‑партнёрами (интеграция)
- Квесты, завязанные на партнёров/ваучеры (переход в Quest)
- Награды Points (отображение в Connect)
