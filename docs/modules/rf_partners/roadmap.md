# Russian Friendly Asia — Roadmap

---

## Этап 1 — MVP (2–3 недели)

### Пользовательская часть
- Каталог партнёров
- Каталог ваучеров
- Страница партнёра
- Страница ваучера
- Поиск и фильтры

### Бэкенд
- Partner Service
- Voucher Service
- Reviews (базовый)
- Привязка к городам Atlas
- API для публичного каталога

## Этап 2 — Кабинет партнёра (3–4 недели)

- Ответы на отзывы
- Базовая статистика

## Этап 3 — PRO Dashboard (3 недели)
