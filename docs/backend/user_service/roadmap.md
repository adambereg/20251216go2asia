# User Service — Roadmap

## Этап 1 — MVP (обязательно для старта)

- Регистрация по email + пароль.
- Логин/лог-аут, refresh-токены.
- Подтверждение email.
- Сброс пароля.
- `/me` + базовый профиль (display_name, avatar, язык).
- Роли: `user`, `admin`.
- JWT-авторизация для других сервисов.

## Этап 2 — Роли и PRO

- Добавить роли `vip`, `pro`, `business_partner_owner`, `moderator`.
- Админ-интерфейсы/эндпоинты для смены роли.
- Поле `visible_in_guru` и базовый набор полей PRO (специализации, языки, почасовая ставка).
- Публичные контакты PRO (telegram, phone, website).
- Интеграция с Guru Service:
  - internal API или события для синхронизации PRO.

## Этап 3 — Социальный граф и приватность

- Follow/friends-механика (`/me/follow`, `/followers`).
- Блокировки (`/me/block`).
- Privacy settings (visibility, show_in_search, show_location).
- Больше настроек уведомлений.

## Этап 4 — Расширенный auth

- OAuth-провайдеры (Google, VK и т.п.).
- Device management (список активных сессий, отключение).
- 2FA (TOTP/SMS) — опционально, при необходимости.

## Этап 5 — Регуляторика и “право на забвение”

- Механика экспорта данных пользователя.
- Полное удаление (по запросу) с контролируемым обезличиванием контента.
