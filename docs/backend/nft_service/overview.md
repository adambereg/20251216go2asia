# NFT Service — Обзор

## Роль NFT в экосистеме Go2Asia

NFT в Go2Asia — это **бейджи достижений и репутации**, а не только «картинки в блокчейне».  
Они:

- показывают опыт путешественника (квесты, города, места, события),
- фиксируют вклад PRO и партнёров (создание квестов, привлечение RF-партнёров),
- могут давать доступ к особому контенту и привилегиям,
- частично выводятся в блокчейн TON как “витрина достижений” наружу.

Чтобы этим управлять, нужен отдельный, хорошо структурированный сервис.

## Назначение NFT Service

**NFT Service** — специализированный микросервис, который:

1. Ведёт **каталог достижений** (типов NFT-бейджей) и условия их получения.
2. Выдаёт **оффчейн NFT-бейджи** пользователям (официальный реестр репутации).
3. Отслеживает **прогресс и уровни** бейджей (Level I, II, III…).
4. Интегрируется с **Connect Service**, Quest Service, Points Service, RF Service и др., чтобы реагировать на события и выдавать/апгрейдить бейджи.
5. Через **Blockchain Gateway Service** инициирует выпуск on‑chain NFT в TON и хранит связь:
   - оффчейн NFTInstance ↔ on‑chain token_id.
6. Предоставляет фронтенду (через API Gateway) готовые данные для:
   - профиля пользователя,
   - витрины достижений PRO,
   - вкладок «Достижения» в Space / Connect.

## Примеры бейджей

- **Explorer I / II / III** — за прохождение 1 / 5 / 20 квестов.
- **City Explorer** — за посещение N разных городов.
- **Local Gourmet** — за отзывы и оценки RF-партнёров.
- **Quest Author / Senior Quest Author** — для PRO, создающих квесты.
- **Partner Curator** — за привлечение RF-партнёров.
- **Premium Voyager** — за использование премиум-ваучеров.
- **Top Referrer** — за реферальную активность (через Referral + Connect).

Каждый бейдж — это **тип NFT** (NFTType) и один или несколько **экземпляров** (NFTInstance) у конкретных пользователей.

## Границы ответственности

NFT Service:

- **ОТВЕЧАЕТ** за:
  - модель и каталог достижений,
  - выдачу, апгрейд и отзыв оффчейн NFT,
  - ончейн-интеграцию (mint/burn через Gateway),
  - хранение истории и аудита,
  - отдачу списка бейджей для UI.

- **НЕ ОТВЕЧАЕТ** за:
  - экономику Points и G2A (Points / Connect / Blockchain Gateway),
  - расчёт наград (какой NFT кому положен) — это Connect Service,
  - UX-рендеринг бейджей (это фронтенд),
  - юридическое и финансовое значение on‑chain токенов (Gateway + юристы).

NFT Service — это **«мозг достижений»**, но не «кошелёк» и не «решатель экономики».
