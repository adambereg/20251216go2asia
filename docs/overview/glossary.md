# Go2Asia — Glossary
## Единый словарь терминов экосистемы Go2Asia  
Версия 1.0

Документ определяет все ключевые термины экосистемы Go2Asia, чтобы унифицировать язык разработки, контента и коммуникации между модулями, микросервисами, фронтендом, backend-командами и AI-агентами.

---

# 1. Основные понятия

### **Go2Asia**
Цифровая экосистема для путешественников, экспатов и бизнеса в Юго-Восточной Азии. Включает Atlas, Pulse, Guru, RF, Quest, Rielt, Space, Blog, Connect и другие модули.

### **PWA Shell**
Главная оболочка приложения: навигация, роутинг, авторизация, интеграция микрофронтендов.

### **Модуль (Module)**
Функциональный блок экосистемы Go2Asia (Atlas, Pulse, Blog, RF и т. д.), реализованный как микрофронтенд и связанный через единый PWA Shell.

### **Микросервис (Service)**
Backend-компонент, отвечающий за конкретную доменную область (Atlas Service, Pulse Service, Rielt Service и т. д.).

---

# 2. Роли пользователей

### **Spacer**
Базовый пользователь платформы. Может пользоваться основным функционалом, выполнять квесты, получать Points.

### **VIP Spacer**
Платный уровень. Имеет доступ к премиальным функциям:  
— покупка премиум-ваучеров,  
— тратит Points,  
— повышенный кешбэк.

### **PRO Spacer**
Эксперт, организатор и партнёр платформы:  
— создаёт квесты,  
— привлекает бизнес-партнеров,  
— отображается в Guru как «человек рядом»,  
— получает вознаграждения Points и G2A.

---

# 3. Основные модули экосистемы

### **Atlas Asia**
Путеводитель по странам, городам и местам ЮВА. Включает историю, культуру, маршруты, локации.

### **Pulse Asia**
Календарь событий: концерты, фестивали, выставки, локальные мероприятия.

### **Blog Asia**
Редакционный и пользовательский контент: статьи, обзоры, подборки, тематические хабы.

### **Guru Asia**
Главный модуль персональных рекомендаций:  
- места рядом,  
- события рядом,  
- PRO-гиды рядом,  
- AI-путеводитель.

### **Russian Friendly**
Каталог дружественных россиянам бизнесов в ЮВА. Ваучеры, премиум-ваучеры, рейтинги, отзывы.

### **Quest Asia**
Система квестов, экскурсий, челленджей и миссий (онлайн и офлайн).

### **Rielt Asia**
Каталог и поиск жилья: аренда, условия, отзывы, фильтры, коммуникация через реакции.

### **Space Asia**
Социальная сеть экосистемы:  
- лента контента,  
- репосты,  
- реакции,  
- ветки общения.

### **Connect Layer (Tokenomics)**
Система вознаграждений, уровнями:
- Points (баллы),  
- NFT (ачивки),  
- G2A (ончейн-токен, фаза 2).

---

# 4. Backend сервисы

### **User Service**
Регистрация, авторизация (Clerk), профили пользователей, роли.

### **Atlas Service**
Страны, города, места, маршруты, точки интереса.

### **Pulse Service**
События, расписания, афиши.

### **Media Service**
Файлы, изображения, обложки, CDN, объектное хранилище.

### **Content Service**
Посты, записи, заметки, публикации Space Asia.

### **Reactions Service**
Единая система взаимодействия с контентом:
- лайки,  
- репосты,  
- short-review,  
- rating,  
- вопросы,  
- ветки общения.

### **Quest Service**
Квесты, чекпоинты, верификация, награды.

### **Rielt Service**
Объявления, жильё, фильтры, статусы, взаимодействие с владельцем через реакции.

### **Voucher Service**
Ваучеры и премиум-ваучеры партнёрских бизнесов.

### **RF Service**
Партнёрские заведения, категории, верификация, статусы.

### **Referral Service**
Реферальная система: рефералы, субрефералы, начисления Points.

### **Connect Service**
Центральный оркестратор токеномики.  
Принимает события → применяет правила → распределяет награды.

### **Points Service**
Учёт баллов, балансы, начисления и списания.

### **NFT Service**
Выдача бейджей, прогресс уровней, ончейн-вывод NFT.

### **Blockchain Gateway Service**
Шлюз для транзакций в TON (mint/burn/transfer G2A, NFT).

### **Notification Service**
Сообщения, push, email, уведомления о событиях.

### **Logging & Analytics Service**
Централизованный сбор логов, трассировка, метрики.

# 5. Токеномика Go2Asia

### **Points**
Оффчейн-баллы за действия пользователя.  
Используются для:
- выполнений квестов,  
- активности,  
- продвижения,  
- вознаграждений PRO.

### **NFT (Achievements)**
Наградные жетоны:
- Explorer,  
- Influencer,  
- Quest Master и т. д.

### **G2A Token**
Ончейн-токен Go2Asia (фаза 2):
- расчёты премиум-ваучеров,  
- PRO-комиссии,  
- вывод на кошелёк пользователя.

---

# 6. Реакции (Reaction Types)

### Основные типы:
- **like** — отметка «нравится»  
- **repost** — перепубликация с опциональным комментарием  
- **rating** — оценка места/жилья/события  
- **short_review** — короткий отзыв  
- **feedback** — текстовое взаимодействие  
- **completed** — завершение квеста  
- **bookmark** — сохранить  
- **question** — задать вопрос  
- **contact_request** — запрос контакта  
- **thread_reply** — ответ в ветке общения  

### Принцип:
**В Go2Asia нет комментариев** — вся коммуникация основана на реакциях и ветках общения.

---

# 7. Ветки общения (Threads)

Используются вместо чатов.

Триггеры создания:
- question  
- contact_request  
- booking inquiry  
- event inquiry  

Ответы — через **thread_reply**.
