# SEO Strategy — стратегия поисковой оптимизации Go2Asia

Документ задаёт правила SEO для всех модулей: Atlas, Pulse, Blog, RF, Guru, Rielt и др.

---

# 1. Основные цели SEO

- рост органического трафика,
- повышение видимости в Google и региональных поисковиках,
- усиление внутренней перелинковки,
- создание контента экспертного качества.

---

# 2. Структура SEO для страниц

Каждая страница должна включать:

1. **Title (до 60 символов)**  
   содержит основное ключевое слово.

2. **Meta Description (150–160 символов)**  
   краткое и точное описание.

3. **H1 — главный заголовок**  
   отражает основной смысл страницы.

4. **H2/H3 структуру**  
   под ключевые темы.

5. **Внутренние ссылки**  
   обязательны для:
   - Atlas → города / места  
   - Blog → Atlas / Pulse  
   - Pulse → локации  

6. **Alt-теги изображений**  
   описательные, но краткие.

---

# 3. Для Atlas

Ключевые слова:

- страна: "[Страна] travel guide", "[Страна] tips", "[куда поехать в [Страна]]"
- город: "what to do in [Город]", "best places in [Город]"
- место: "[Place] review", "[Place] info"

---

# 4. Для Pulse

- "[событие] + [город]"
- "[что делать] + [дата]"
- "[концерты, фестивали, встречи] + [город]"

---

# 5. Для Blog

- длинные ключевые фразы,
- H2 разбивают смысл на требования поведения пользователя,
- цель — быть полноценным экспертным гайдом.

---

# 6. Техническое SEO

- чистые URL,
- быстрая загрузка,
- отсутствие тяжёлых изображений,
- корректные OpenGraph и Twitter-карточки.

---

# 7. AI-процессинг

AI-агенты должны:

- генерировать Title/Description автоматически,
- соблюдать структуру,
- понимать интент пользователя,
- избегать переспама.
