# Blog Editorial Guide — редакционный стандарт для Blog Asia

Blog Asia — медиа Go2Asia: статьи, аналитика, гайды, обзоры, интервью.

---

# 1. Форматы статей

- **Гайд** (как, где, когда)
- **Разбор / аналитика**
- **Маршрут / подборка**
- **Интервью**
- **История места**
- **Лайфхаки**

---

# 2. Структура статьи

1. **Hero + заголовок**
2. **Лид (2–3 строки)**
3. **Основные разделы** (H2)
4. **Подразделы** (H3)
5. **Списки / блоки фактов**
6. **Заключение**

---

# 3. Требования к стилю

✔ аналитичность, полезность  
✔ структурированность  
✔ достоверность  
✔ отсутствие воды  
✔ отсутствие эмоций  

✖ нельзя использовать разговорный стиль  
✖ нельзя давать субъективные оценки  
✖ нельзя писать рекламные тексты  

---

# 4. Длина статьи

- 3 000–8 000 символов
- Чёткая структура, без бесполезных абзацев

---

# 5. SEO для Blog

- h1 — основное ключевое слово  
- h2/h3 — расширяющие ключи  
- обязательно:  
  - внутренние ссылки на Atlas, Pulse, RF  
  - мета-описание 150–160 символов  

---

# 6. Иллюстрации

- формат 16:9  
- изображения высокого качества  
- не использовать изображения с водяными знаками
