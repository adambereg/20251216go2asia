# Writing Guidelines — правила написания контента для Go2Asia

Этот документ определяет стандарты написания текстов для всех модулей Go2Asia: Atlas, Pulse, Blog, Guru, RF, Rielt и др. Правила обязательны для всех AI-агентов, редакторов и контент-менеджеров.

---

# 1. Философия контента Go2Asia

Контент должен быть:

- **ясным** (простым для чтения),
- **структурированным**,
- **полезным для путешественника/экспата**,
- **конкретным и фактическим**,
- **без воды, клише и рекламных оборотов**,
- **дружелюбным и живым**, но без лишней эмоциональности.

Главное правило:  
> Пользователь должен за 5 секунд получить то, что ему важно.

---

# 2. Стиль

## 2.1. Короткие предложения  
Средняя длина — 8–14 слов.

## 2.2. Никаких сложных оборотов  
Избегать канцелярита, неопределённости, длинных вводных.

## 2.3. Минимум прилагательных  
Никакой "потрясающий пляж", "великолепная кухня".  
Только факты: "тихий пляж", "местная кухня с упором на морепродукты".

## 2.4. Никаких оценочных суждений  
✔ "Пляж подходит для спокойного отдыха"  
✖ "Это лучший пляж в регионе"

## 2.5. Актуальность  
Информация должна быть:
- проверяемой,
- обновляемой,
- датируемой (когда важно).

---

# 3. Форматы текста

## 3.1. Короткие абзацы  
1 абзац = 2–3 строки.

## 3.2. Списки  
Использовать списки, когда:
- есть перечисление,
- есть факты,
- есть шаги.

## 3.3. Заголовки  
Структура документа должна быть иерархичной.

---

# 4. Тональность

- дружественная,
- информативная,
- уверенная,
- без сюсюканья,
- без субъективности.

---

# 5. SEO-правила (важно!)

- заголовки должны содержать ключевые слова,
- использовать подзаголовки и списки,
- избегать «keyword stuffing»,
- финальный абзац должен быть обобщающим.

---

# 6. Запрещено

- длинные поэтические описания,
- рекламные тексты,
- субъективная критика,
- распространение неверной информации,
- копирование чужих текстов.

---

# 7. Обязательные правила для AI-агентов

AI должен:

- проверять факты,
- сохранять структуру,
- избегать повторов,
- не использовать эмоциональные приёмы маркетинга,
- не генерировать длинные фразы без смысла.
