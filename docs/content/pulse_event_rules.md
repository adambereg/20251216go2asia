# Pulse Event Rules — стандарты создания событий для Pulse Asia

Pulse — календарь событий ЮВА: концерты, фестивали, занятия, встречи, медитации, йога, бизнес-мероприятия и др.

Контент должен быть фактическим, чётким и регулярно обновляться.

---

# 1. Типы событий Pulse

- фестивали,
- образовательные мероприятия,
- духовные практики,
- концерты,
- клубные встречи,
- семейные и детские события,
- спортивные события,
- бизнес-митапы.

---

# 2. Структура карточки события

1. **Название события**  
   до 60 символов, без эмоций.

2. **Фото обложки**  
   формат 4:3.

3. **Краткое описание**  
   1 абзац, что будет происходить.

4. **Дата и время**  
   всегда указывать часовую зону.

5. **Локация**  
   точный адрес + район.

6. **Статус**  
   - Бесплатно  
   - Платно  
   - Предварительная регистрация  

7. **Ссылка на оригинал / регистрацию**

---

# 3. Структура страницы события

1. Hero  
2. Описание  
3. Программа (если есть)  
4. Организатор  
5. Цена и условия участия  
6. Как добраться  

---

# 4. Стандарты текста

✔ Фактология  
✔ Конкретика: кто, что, где, когда  
✔ Краткость  

✖ Нет эмоциональных эпитетов  
✖ Нет субъективных оценок  
✖ Нет маркетинговых фраз  

---

# 5. Требования к AI-агентам

- автоматическая проверка даты и валидности ссылки,
- генерация структурированного описания,
- корректная классификация типа события,
- запрещено придумывать имена, организаторов, цены.
