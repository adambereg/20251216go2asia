# Atlas Content Rules — правила создания контента для Atlas Asia

Atlas — энциклопедия стран, городов и мест ЮВА.  
Контент должен быть точным, полезным и структурированным.

---

# 1. Типы страниц Atlas

1. Страна  
2. Город  
3. Место (пляж, храм, район, рынок, достопримечательность, природный объект)  
4. Тематическая подборка (если появится)

---

# 2. Структура страницы "Страна"

1. **Обзор**  
   2–3 абзаца о стране: география, культура, особенности.

2. **История**  
   Кратко, только ключевые точки.

3. **География**  
   Крупные регионы, климат, особенности.

4. **Культура**  
   Основные традиции, религии, особенности людей.

5. **Проживание**  
   Где жить, виды жилья, особенности цен.

6. **Визы**  
   Факты, без упоминаний цен (если они меняются — дата обязательна).

7. **Бизнес**  
   Особенности предпринимательства, налоги, сферы.

---

# 3. Структура страницы "Город"

1. **Обзор города**  
2. **Районы**  
3. **Транспорт**  
4. **Погода**  
5. **Особенности жизни**  
6. **Основные места и достопримечательности**

---

# 4. Структура страницы "Место"

1. Название + короткое описание  
2. Где находится (район, расстояние)  
3. Что интересного  
4. Когда лучше приезжать  
5. Цена/время посещения (если применимо)  
6. Советы  

---

# 5. Стандарты подачи информации

- короткие абзацы,
- факты,
- полезная информация для туристов и экспатов,
- никаких субъективных оценок.

---

# 6. Фото

Atlas должен иметь:

- качественные изображения,
- формат 16:9,
- фото без людей (по возможности),
- яркие, информативные примеры.

---

# 7. Запрещено

- писать «это лучшее место/город/страна»,
- добавлять байки, легенды, субъективные рассказы,
- преувеличивать.
